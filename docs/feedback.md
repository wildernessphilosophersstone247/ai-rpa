# Feedback

We welcome feedback on the public `aivanelabs/ai-rpa` Android REPL materials.

## How to share feedback

- Open an issue or discussion on `https://github.com/aivanelabs/ai-rpa`.
- Include device info and CLI command snippets when reporting behavior.
- Mention whether the feedback concerns docs, the Python client, or sample skills.

## Quick questions we’re asking

1. Are the quickstart and smoke flows clear enough for first-time operators?
2. Do the sample skill prompts capture real agent needs?
3. What other references or matrices should we publish before launch?
