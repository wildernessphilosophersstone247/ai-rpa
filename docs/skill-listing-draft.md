# Skill Listing Draft

## Title

AIVane Android REPL

## One-line Summary

Let AI agents inspect and control an Android phone step by step over LAN.

## Short Description

Use AIVane to connect to an Android device, list launcher apps, inspect the current UI tree, and perform tap/input/swipe/back/home/screenshot actions through a REPL-friendly workflow.

## Audience

- Codex users
- Claude Code users
- OpenClaw users
- Agent builders experimenting with mobile control

## Positioning

- Beta / experimental
- LAN-only workflow
- REPL-first experience
- Advanced template execution remains available as a compatibility path

## Known Limits

- Accessibility must be enabled manually
- Screenshot requires explicit authorization on device
- The current release is aimed at exploration and testing, not fully unattended production use

## Call To Action

Install the Android app, connect from your desktop over LAN, run the smoke flow, and report what breaks or feels awkward.
